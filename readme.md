# Anti-Paotu  
一个简单的反跑图插件，支持Onebot通知。

## 权限和命令  
**权限**  
* antipaotu.notice: 接收跑图检测通知的权限（默认：OP）  
* antipaotu.bypass: 跳过跑图检测的权限（默认：无）  
* antipaotu.admin: 管理插件的权限（默认：OP）  

**命令**  
* /antipaotu reload: 重载插件配置  
* /antipaotu status <玩家名>: 查看玩家的检测状态  
* /antipaotu unban <玩家名>: 解除玩家的临时封禁
